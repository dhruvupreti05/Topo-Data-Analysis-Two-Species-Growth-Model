\subsection{How to see you current working directory}

Every file and folder (directory) has an associated path which tells the operating system where it is located. To see the path of the directory you are currently in type:

\begin{bash}
    eshasury@MacBook-Pro: -% pwd
    /Users/eshasury
\end{bash}

This means that the user is currently in the directory \verb+eshasury+ in the directory \verb+Users+.
